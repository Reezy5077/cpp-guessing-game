# Number Guessing Game (C++)

This is a simple console game written in C++.

## How it works:
The computer chooses a random number between 1 and 100.
The player tries to guess the number.

## Features:
- Random number generation
- Input handling
- Loop and conditions

## What I learned:
- C++ basics
- if/else statements
- loops (do-while)
- working with random numbers

## Purpose:
This project was created to improve my programming skills and to show my interest in IT and software development ;)
